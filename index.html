About this Class

Course: Web Design & Development — One-page portfolio project.

Students were asked to create a one-page portfolio using HTML and CSS. Requirements: a header with links for Home, About, and Gallery; Home must include a hero section; About must describe the class; Gallery must show each student with a photo, name, and dream job. The site will be deployed through GitHub Pages and students will submit the live URL and the GitHub repository link.
